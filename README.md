# Módulo de Instalaciones Civiles e Industriales

Proyecto Quarto Book para publicar en GitHub Pages.

## Renderizar localmente

```bash
quarto render
```

## Publicar en GitHub Pages

1. Crear un repositorio llamado `modulo-instalaciones-civiles-industriales`.
2. Subir todos los archivos del proyecto.
3. Ejecutar:

```bash
quarto render
git add .
git commit -m "Actualizar módulo de instalaciones civiles e industriales"
git push
```

4. En GitHub: Settings → Pages → Deploy from a branch → main → /docs.
